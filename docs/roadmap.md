# Roadmap

## Level 1: Basic Demo

- minimal front-end page
- pull request template
- CODEOWNERS
- CI for lint and build

## Level 2: Better Review Practice

- conventional commits
- branch naming guide
- screenshots required in PRs for UI changes
- review checklist by risk category

## Level 3: Stronger CI

- add unit tests
- add preview deployment
- add dependency audit
- add auto-labeling for pull requests

## Level 4: Team Workflow

- protected environments
- release tags
- changelog automation
- merge queue
